# Visionary Innovation Hub - Website

## Overview
A modern, interactive website for an innovation organization featuring multiple pages with JavaScript functionality and responsive design.

## Quick Start
1) Download all the files
2) Open index.html in any web browser available in your computer
3) Use the navigation menu to explore all the pages
